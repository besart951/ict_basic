# Lesson: Listen

## Einfache Erklärung
Listen speichern mehrere Werte in einer Reihenfolge.

## Beispielcode
```python
farben = ["rot", "blau", "grün"]
print(farben[0])
```

## Visualisierung
Index: 0 1 2
Wert:  rot blau grün

## Kleine Aufgaben
1. Erstelle eine Liste mit 4 Tieren.
2. Gib das zweite Tier aus.
3. Füge ein neues Tier hinzu.
