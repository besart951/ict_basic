# Programmierkurs (12–15 Jahre) mit VS Code

Willkommen! Dieser Kurs führt dich **Schritt für Schritt** durch:

1. **Git-Grundlagen** – Änderungen speichern und teilen
2. **Python-Grundlagen** – Programmieren lernen
3. **Objektorientierte Programmierung (OOP)** – Klassen und Objekte
4. **Eigenes Text-RPG im Terminal** – Alles zusammen

## Voraussetzungen

- VS Code
- Python 3.10+
- Git

## VS Code einrichten

1. Öffne den Ordner `programmierkurs` in VS Code.
2. Installiere die Erweiterungen:
   - **Python** (Microsoft)
   - **Pylance**
3. Wähle den Python-Interpreter:
   - `Ctrl+Shift+P` → `Python: Select Interpreter`

## Kursstruktur

- `01_git` – Git lernen (Commit, Push, Pull mit VS Code)
- `02_python` – Python-Bausteine
- `03_oop` – Klassen, Attribute, Methoden
- `04_spiel` – komplettes RPG-Projekt

## Python-Dateien ausführen

Im Terminal in VS Code (im passenden Ordner):

```
python dateiname.py
```

Viel Erfolg und viel Spass beim Coden! 🚀
