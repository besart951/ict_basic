# Phase 4: Abschlussprojekt – Text-RPG

Jetzt baust du ein kleines Spiel im Terminal.

## Ziel
Ein spielbares RPG mit den Klassen:
- `Player`
- `Monster`
- `BattleSystem`
- `Game`

## Ablauf
1. Spielaufbau verstehen (`01_spielaufbau`)
2. Spieler bauen (`02_spieler`)
3. Monster bauen (`03_monster`)
4. Kampfsystem bauen (`04_kampfsystem`)
5. Finales Spiel starten (`abschlussprojekt/dungeon_game.py`)
