# Lesson: Kampfsystem

## Einfache Erklärung
Das Kampfsystem rechnet eine Kampfrunde: Spieler greift an, dann Monster.

## Beispielcode
```python
battle = BattleSystem()
battle.fight_round(player, monster)
```

## Visualisierung
Spieler-Angriff -> Monster-Angriff -> neue HP anzeigen
