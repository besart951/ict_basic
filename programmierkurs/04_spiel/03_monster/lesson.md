# Lesson: Monster-Klasse

## Einfache Erklärung
Monster funktioniert ähnlich wie Player, aber mit anderen Werten.

## Beispielcode
```python
monster = Monster("Skelett")
```

## Visualisierung
Monster wird erzeugt -> kämpft -> kann besiegt werden

## Kleine Aufgaben
1. Erstelle zwei verschiedene Monster.
2. Gib deren Werte aus.
