# Übung: Repository erstellen

## Aufgabe

1. Gehe auf [github.com](https://github.com) und erstelle ein neues Repository mit dem Namen `mein-erstes-repo`.
2. Öffne VS Code und erstelle einen neuen Ordner `mein-erstes-repo` auf deinem Computer.
3. Öffne diesen Ordner in VS Code.
4. Öffne die Quellcodeverwaltung (`Ctrl+Shift+G`).
5. Klicke auf **Repository initialisieren**.
6. Verbinde den Ordner mit deinem GitHub-Repository (Remote hinzufügen).

## Ergebnis

Du hast ein lokales Repository, das mit GitHub verbunden ist.
