# Übung: Push zu GitHub

## Aufgabe

1. Stelle sicher, dass du mindestens einen Commit gemacht hast.
2. Öffne die Quellcodeverwaltung in VS Code (`Strg+Shift+G`).
3. Klicke auf **...** (oben rechts in der Quellcodeverwaltung).
4. Wähle **Push**.
5. Gehe auf GitHub und prüfe, ob deine Dateien dort erscheinen.

## Ergebnis

Deine Dateien sind jetzt auf GitHub sichtbar!
