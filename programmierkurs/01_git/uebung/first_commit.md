# Übung: Erster Commit

## Aufgabe

1. Erstelle eine neue Datei `notizen.txt` in deinem Repository-Ordner.
2. Schreibe einen Satz hinein, z. B. `Mein erster Git-Commit!`.
3. Öffne die Quellcodeverwaltung in VS Code (`Strg+Shift+G`).
4. Klicke auf **`+`** neben `notizen.txt`, um die Datei zu stagen.
5. Gib die Commit-Nachricht `Start` ein.
6. Klicke auf **Commit ✓**.

## Ergebnis

Dein erster Commit ist gespeichert!
