# Dein erster Commit

## Schritt-für-Schritt in VS Code

### 1. Repository auf GitHub erstellen
1. Gehe auf [github.com](https://github.com) und melde dich an.
2. Klicke auf **New** und erstelle ein neues Repository.
3. Kopiere den Repository-Link.

### 2. Ordner in VS Code mit GitHub verbinden
1. Öffne den Ordner in VS Code.
2. Klicke auf `Quellcodeverwaltung` (`Ctrl+Shift+G`).
3. Klicke auf **Repository initialisieren**.
4. Klicke auf **...** (oben rechts) → **Remote hinzufügen**.
5. Füge den GitHub-Link ein und bestätige.

### 3. Commit erstellen
1. Ändere eine Datei oder erstelle eine neue.
2. Öffne die Quellcodeverwaltung (`Ctrl+Shift+G`).
3. Klicke auf **`+`** neben der geänderten Datei.
4. Gib eine Nachricht ein, z. B. `Erster Commit`.
5. Klicke auf **Commit ✓**.

### 4. Push (Hochladen)
1. Klicke auf **...** in der Quellcodeverwaltung.
2. Wähle **Push**.
3. Prüfe auf GitHub, ob die Dateien angekommen sind.

### 5. Pull (Herunterladen)
1. Klicke auf **...** in der Quellcodeverwaltung.
2. Wähle **Pull**.
3. Deine Dateien werden aktualisiert.
