# Herausforderung: Git-Zeitmaschine

Du sollst drei verschiedene Versionen einer Datei erstellen und sie committen.

## Aufgabe

1. Erstelle eine Datei `geschichte.txt` in deinem Repository-Ordner.
2. Schreibe einen Satz hinein und mache einen Commit (z. B. `Version 1`).
3. Ändere den Text und mache einen zweiten Commit (z. B. `Version 2`).
4. Ändere den Text nochmals und mache einen dritten Commit (z. B. `Version 3`).
5. Pushe alle Commits zu GitHub.
6. Öffne GitHub und schaue dir die Commit-Liste an.

## Ergebnis

Du kannst auf GitHub alle drei Versionen deiner Datei sehen.

## Bonus

Erstelle eine zweite Datei `ideen.txt` und committe sie separat.
