# Lesson: Attribute

## Einfache Erklärung
Attribute sind Eigenschaften eines Objekts.

## Beispielcode
```python
class Player:
    def __init__(self, name, hp):
        self.name = name
        self.hp = hp
```

## Visualisierung
Player -> name, hp

## Kleine Aufgaben
1. Füge `level` hinzu.
2. Gib alle Attribute aus.
