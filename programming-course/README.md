# Programmierkurs (12–15 Jahre) mit VS Code

Willkommen! Dieser Kurs führt dich **Schritt für Schritt** durch:

1. **Git-Grundlagen**
2. **Python-Basics**
3. **Objektorientierte Programmierung (OOP)**
4. **Eigenes Text-RPG im Terminal**

## Voraussetzungen

- VS Code
- Python 3.10+
- Git

## VS Code einrichten

1. Öffne den Ordner `programming-course` in VS Code.
2. Installiere die Extensions:
   - **Python** (Microsoft)
   - **Pylance**
   - **GitLens** (optional, hilfreich für Git)
3. Wähle den Python-Interpreter:
   - `Strg+Shift+P` → `Python: Select Interpreter`
4. Optional: Auto-Formatierung aktivieren:
   - Extension: **Black Formatter**
   - In den Einstellungen: `Format On Save` einschalten

## Kursstruktur

- `01_git` – Git lernen (init, add, commit, push, pull)
- `02_python` – Python-Bausteine
- `03_oop` – Klassen, Attribute, Methoden
- `04_game` – komplettes RPG-Projekt

## Ausführen

Im Terminal in VS Code (im passenden Ordner):

```bash
python dateiname.py
```

Viel Erfolg und viel Spaß beim Coden! 🚀
