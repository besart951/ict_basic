# Challenge: Git-Zeitmaschine

Du sollst drei Versionen einer Datei bauen und zurückspringen.

## Aufgabe

1. Erstelle `story.txt`.
2. Mache 3 Commits mit unterschiedlichen Texten.
3. Nutze `git log --oneline`.
4. Springe auf einen älteren Commit (mit `git checkout <hash>`).
5. Erkläre, was mit der Datei passiert ist.

## Bonus

Erstelle einen neuen Branch `experiment` und probiere dort eine alternative Story.
