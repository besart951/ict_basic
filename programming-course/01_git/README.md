# Phase 1: Git (Versionskontrolle)

In dieser Phase lernst du, wie du deine Projekte sicher speicherst und Änderungen nachvollziehst.

## VS Code Setup (kurz)

- Öffne den `programming-course`-Ordner.
- Installiere **GitLens** (optional).
- Prüfe im Terminal:
  ```bash
  git --version
  ```

## Lernziele

- `git init`
- `git add`
- `git commit`
- `git push`
- `git pull`

Arbeite die Dateien in Reihenfolge durch:
1. `01_git_basics.md`
2. `02_first_commit.md`
3. Übungen in `exercise/`
4. Challenge in `challenge/`
