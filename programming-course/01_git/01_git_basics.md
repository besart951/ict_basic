# Git Basics

Git ist wie eine Zeitmaschine für deinen Code.

## Wichtigste Befehle

```bash
git init
git status
git add .
git commit -m "Meine Nachricht"
```

## Visualisierung

Arbeitsordner -> Staging Area -> Repository

`Datei ändern` -> `git add` -> `git commit`

## Mini-Aufgaben

1. Erstelle einen neuen Ordner `mein_test_repo`.
2. Führe `git init` aus.
3. Erstelle eine Datei `notizen.txt`.
4. Prüfe den Status mit `git status`.
