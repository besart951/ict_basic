# Lesson: Bedingungen (if)

## Einfache Erklärung
Mit `if` entscheidest du, welcher Code ausgeführt wird.

## Beispielcode
```python
punkte = 12
if punkte >= 10:
    print("Bestanden")
else:
    print("Nicht bestanden")
```

## Visualisierung
Bedingung wahr? -> Ja: Block A, Nein: Block B

1. Prüfe, ob jemand volljährig ist.
2. Prüfe, ob eine Zahl gerade ist.
