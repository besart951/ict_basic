# Lesson: Funktionen

## Einfache Erklärung
Funktionen sind wiederverwendbare Code-Blöcke.

## Beispielcode
```python
def begruessung(name):
    print(f"Hallo {name}!")
```

## Visualisierung
Aufruf -> Funktion läuft -> Ergebnis
