# Lesson: Variablen

## Einfache Erklärung
Variablen speichern Daten, z. B. Namen oder Zahlen.

## Beispielcode
```python
name = "Lina"
alter = 13
print(name, alter)
```

## Visualisierung
`name` -> "Lina"
`alter` -> 13
